# Activity Diagrams & BPMN - COBIT 2019 Design Toolkit

**Versi Dokumen:** 1.0  
**Tanggal:** Desember 2024

---

## 1. Pendahuluan

Dokumen ini menjelaskan alur proses bisnis (Business Process) dalam aplikasi COBIT 2019 Design Toolkit menggunakan notasi BPMN (Business Process Model and Notation) dan Activity Diagram.

### 1.1 Notasi yang Digunakan

| Simbol | Nama | Deskripsi |
|--------|------|-----------|
| ○ | Start Event | Titik mulai proses |
| ◉ | End Event | Titik akhir proses |
| □ | Task/Activity | Aktivitas yang dilakukan |
| ◇ | Gateway (Decision) | Titik keputusan/percabangan |
| → | Sequence Flow | Alur proses |
| ⬚ | Pool/Lane | Swimlane untuk aktor |

## 5. Appendix

### 5.1 Glossary BPMN

| Term | Definition |
|------|------------|
| Pool | Container untuk satu participant/aktor dalam proses |
| Lane | Subdivisi dalam pool untuk mengorganisasi aktivitas |
| Task | Unit kerja atomic yang tidak dapat dipecah lagi |
| Gateway | Titik keputusan atau penggabungan alur |
| Event | Sesuatu yang terjadi dalam proses (start, end, intermediate) |
| Sequence Flow | Menunjukkan urutan aktivitas |

### 5.2 Level Achievement Codes

| Code | Description |
|------|-------------|
| F | Fully Achieved (85-100%) |
| L | Largely Achieved (50-84%) |
| P | Partially Achieved (15-49%) |
| N | Not Achieved (0-14%) |
