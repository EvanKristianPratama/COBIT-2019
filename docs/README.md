## Ikhtisar Aplikasi

COBIT 2019 Design Toolkit adalah aplikasi berbasis web yang membantu organisasi dalam mengimplementasikan framework COBIT 2019 untuk governance dan manajemen teknologi informasi. Aplikasi ini menyediakan:

- **Design Factor Assessment** - Penilaian 10 faktor desain COBIT 2019
- **Objective Assessment** - Evaluasi objectives dan practices COBIT
- **Assessment Evaluation** - Evaluasi aktivitas berdasarkan COBIT framework

## Teknologi yang Digunakan

- **Backend**: Laravel (PHP)
- **Database**: MySQL
- **Frontend**: Blade Templates + Bootstrap
- **Authentication**: Laravel Auth + Google OAuth
